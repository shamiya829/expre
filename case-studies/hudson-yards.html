Case study • Mixed-use

Hudson Yards

Icon + programming + tenant mix used to create a new district and sustain demand.

Mixed-use playbook Circulation principle Back to library

At a glance

  • Experience strategy District creation + landmarks + events
  • Tenant mix Retail + dining + office + hospitality
  • Design elements Public realm sequencing, icons, nodes
  • Financial logic Footfall + premium positioning
Developer lessons

What to steal

  • Use magnets (icons, venues) to anchor circulation.
  • Design clear “district moments” that orient visitors.
  • Program the public realm like an event venue.
  • Balance destination energy with everyday usability.

Background

Hudson Yards is a large-scale mixed-use district where placemaking and destination magnets were used to generate demand at scale.

Experience strategy

Combine landmark moments with a curated mix so the district feels intentional. Magnets create first visits; programming creates return visits.

Tenant mix

Use a ladder: daily uses, destination dining, retail discovery, hospitality, and office—all connected through legible circulation.

Design elements

  • Nodes that concentrate activity (and therefore monetization).
  • Clear vertical circulation to move between levels and uses.
  • Wayfinding language that reinforces “district” not “building.”

Financial logic

Footfall and brand halo support leasing, tenant sales, and premium rents. Public realm investments act as demand infrastructure.

Lessons developers can apply

  • Magnet without flow is wasted; design the loop first.
  • Underwrite the operating system: programming, safety, and service standards.
  • Make the district usable on non-event days (comfort and daily rhythm).
